Deploying ALP Dolomite interactively using Agama

 

This article describes how to deploy SUSE ALP Dolomite (ALP Dolomite) interactively using the Agama installer. To enhance this ALP Dolomite documentation, find its sources at https://github.com/SUSE/doc-modular/edit/main/xml/.

WHAT?

There are several methods you can deploy ALP Dolomite. The Agama installer presents an interactive and user friendly way to achieve this.

WHY?

To give a detailed overview of the whole ALP Dolomite deployment process and related contextual information.

EFFORT

Although the deployment process takes less than 30 minutes, to learn more about ALP Dolomite, its deployment methods, and related considerations and additional configurations takes longer.

REQUIREMENTS

You need to have basic experience with Linux system installation to successfully deploy ALP Dolomite interactively using Agama.
